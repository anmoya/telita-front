# __APP_NAME__

Base reusable de Telita para proyectos nuevos.

## Incluye
- App shell base
- Primitives UI (`Button`, `Input`, `Select`, `Alert`)
- Hook async base (`useAsyncAction`)
- Estructura de carpetas `shared/` y `modules/`
- Guardrails de arquitectura sugeridos

## Comandos
```bash
npm install
npm run dev
npm run lint
npm run typecheck
```

## Variables
Copiar `.env.example` a `.env.local`.
